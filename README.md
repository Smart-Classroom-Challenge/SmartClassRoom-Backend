# SmartClassRoom-Backend
Rest Django Backend

## Run Django Server
1. `cd SmartClassRoom`
2. `python manage.py runserver`

## Test REST API
1. `cd SmartClassRoom`
2. `python manage.py test`

## Django Admin Dashboard

## Database Migration after Model Changes

1. `python manage.py makemigrations`
2. `python manage.py migrate`
